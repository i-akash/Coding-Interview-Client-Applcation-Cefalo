# Coding Interview Client Application

#### Run Instruction

* `git clone https://github.com/i-akash/Coding-Interview-Client-Applcation-Cefalo.git`
* `cd Coding-Interview-Client-Applcation-Cefalo`
* `npm install`
* `npm start`



## Docs

#### Create Problem Tab

- Input Constraint ( `format : json` )

  - Example

    ```json
    {
      "arrayLength": "int",
      "array": "[string]"
    }
    ```

    ```json
    {
      "str1": "string",
      "str2": "string"
    }
    ```

    ```json
    {
      "a": "float",
      "b": "int"
    }
    ```
